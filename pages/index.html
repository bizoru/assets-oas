Inclusion de Estilos en Proyectos OAS

Para poder utilizar los estilos que se presentan a continuacion debera vincular los siguientes estilos al inicio de su archivo layout:

  • bootstrap.min.css
  • metisMenu.min.css
  • timeline.css
  • oas-style.css (Estilos ajustados OAS)
  • morris.css
  • font-awesome.min.css

<!-- Core de Bootstrap -->
<link href="oas/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- MetisMenu CSS -->
<link href="oas/metisMenu/dist/metisMenu.min.css" rel="stylesheet">

<!-- Timeline CSS -->
<link href="dist/css/timeline.css" rel="stylesheet">

<!-- Custom OAS CSS -->
<link href="dist/css/oas-style.css" rel="stylesheet">

<!-- Morris Charts CSS -->
<link href="oas/morrisjs/morris.css" rel="stylesheet">

<!-- Custom Fonts -->
<link href="oas/font-awesome/css/font-awesome.min.css" rel="stylesheet" type="text/css">

Inclusion de scripts Proyectos OAS

Al final de su archivo layout incluya los siguientes scripts de javascript: 


<!-- HTML5 Shim and Respond.js IE8 support of HTML5 elements and media queries -->
<!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
<!--[if lt IE 9]>
<script src="https://oss.maxcdn.com/libs/html5shiv/3.7.0/html5shiv.js"></script>
<script src="https://oss.maxcdn.com/libs/respond.js/1.4.2/respond.min.js"></script>
<![endif]-->

<!-- jQuery -->
<script src="oas/jquery/dist/jquery.min.js"></script>

<!-- Bootstrap Core JavaScript -->
<script src="oas/bootstrap/dist/js/bootstrap.min.js"></script>

<!-- Metis Menu Plugin JavaScript -->
<script src="oas/metisMenu/dist/metisMenu.min.js"></script>

<!-- Morris Charts JavaScript -->
<script src="oas/raphael/raphael-min.js"></script>
<script src="oas/morrisjs/morris.min.js"></script>
<script src="js/morris-data.js"></script>

<!-- Custom Theme JavaScript -->
<script src="dist/js/oas-style.js"></script>

Inclusion de componentes Angular 1.5.7 Proyectos OAS

Para incluir los componentes angular utilizados por defecto en cualquier proyecto, incluya las siguientes lineas en su aplicacion: 


  <script src="oas/angular/angular.min.js"></script>
  <script src="oas/angular-components/components.js"></script>
