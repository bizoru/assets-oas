Para cambiar el color de la barra de progreso utilice las clases CSS progress-bar-danger progress-bar-info progress-bar-success progress-bar-warning

El valor aria-valuenow representa el valor actual de la barra de progreso

El valor mínimo se controla con el atributo aria-valuemin y el valor máximo con el atributo aria-valuemax, sin embargo la apariencia de la barra se controla con el atributo style donde se determina el width (ancho) o progreso de la barra, si desea una barra al 20% debera colocar el atributo style="width: 20%"
